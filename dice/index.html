On Base Dice Roller

Use these dice with the cards and dice version of On Base, a tabletop baseball game based on advanced statistics. The value of the blue die determines whether the pitcher or batter has control of the at-bat. If the value is 1-3 then read the outcome of the 2 d6 dice off the batter columns. If the value is 4-6 then read the outcome of the 2 d6 dice off the pitcher columns.
